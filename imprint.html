This site and its source code are hosted for free on GitHub Pages. It was built with Jekyll version 3.2.1, and is managed with Forestry.io.

It incorporates the Raleway typeface, hosted by Google Fonts and modified Sass code from the Bootstrap framework.

It was developed with Sublime Text 3 for editing code, Safari for previewing and debugging, and SourceTree for version control.

It's been screened for visibility with colourblindness using the Sim Daltonism application by Michel Fortin, and evaluated for accessibility using the WAVE Accessibility Tool.

Email addresses have been obfuscated using the West Bay Web email encoder.
